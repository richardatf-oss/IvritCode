What is IvritCode?

For Developers & Researchers

Under the hood, IvritCode is:

  • A small virtual machine where each letter has precise semantics (see src/vm.ts).
  • Built on finite-field arithmetic (see src/field.ts) with Poseidon-inspired round functions (see src/round.ts).
  • Designed so its execution trace can be expressed as an AIR (Arithmetic Intermediate Representation), suitable for zk-VMs and proof systems.

To run locally: 

npm install
npm run build
npm start

Start with src/index.ts to see the demo program, or open src/vm.ts to add new letters and instructions.
