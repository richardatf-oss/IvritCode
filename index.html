What’s happening on this page?

You’re looking at a tiny machine built out of Hebrew letters. Each letter is treated like an instruction. You pick a few starting numbers, and the letters go to work on them.

Demo program · the letters that run on your numbers
א → ב → ג → ד → סבב

No background in math or coding is needed. Just think of it like watching gears turn:

  • א quietly prepares the machine & “locks in” this run.
  • ב adds your first two numbers.
  • ג multiplies them.
  • ד stirs the values together.
  • סבב performs a deep cryptographic-style mix.

Everything happens right in your browser: no signup, no wallet, nothing saved. Just letters and numbers dancing for a moment.

How to read a line:
t=3 – step number in the run
pc – where we are in the letter sequence
op – which letter ran at this step
regs=[R0,R1,R2,R3] – the four visible numbers
c, h, z – hidden numbers the machine keeps for internal consistency & cryptography.

If You’re Just Curious

At a simple level, IvritCode is a way of asking:

  • What if the Aleph-Bet itself was a tiny computer?
  • What if a story could also be a set of instructions?
  • What happens when letters push numbers around?
  • No background knowledge required.
  • You can’t break anything by experimenting.
  • Same letters + same numbers = same result, every time.

Future versions of this site will include a visual playground where you can drag letters onto a line, press play, and watch the machine animate without ever seeing raw numbers.

If You’re a Developer or Researcher

Under the hood, this demo is driven by a small virtual machine in TypeScript:

  • Each letter in the demo has precise semantics in src/vm.ts.
  • Arithmetic is done in a finite field (src/field.ts).
  • The deep mixing step סבב uses a Poseidon-inspired round (src/round.ts).

The VM is structured so its execution trace can be expressed as an AIR (Arithmetic Intermediate Representation), making it natural to embed inside zk-VMs and proof systems.

Letters Used in This Demo

This first version uses a short alphabet of operations. More letters (and niqqud/trope modifiers) will be added over time.

Letter Name What it does in the machine
א Aleph Leaves your visible registers as they are, but updates hidden state so this run is cryptographically tied to this program and position in the code.
ב Bet Adds the first two registers: R0 = R0 + R1.
ג Gimel Multiplies the first two registers: R0 = R0 × R1.
ד Dalet Stirs R0, R1, R2 together using a linear combination, leaving R3 unchanged.
סבב Sovev Applies a Poseidon-style round to [R0,R1,R2,R3,C], giving a cryptographic permutation of the state.
